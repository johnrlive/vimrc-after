# To install just vim settings

```
cd ~
git clone https://github.com/johnrlive/vimrc-after.git
cd ~/vimrc-after
ln -sf `pwd`/vimrc.local ~/.vimrc.local
ln -sf `pwd`/vimrc.bundles.local ~/.vimrc.bundles.local
```
